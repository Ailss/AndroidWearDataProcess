##中文简介与使用说明
###这是一个对手表数据进行后续处理的工程。
###内容说明：
- mainForCalibration.m: 校准加速度器，得到校准矩阵
- mainForIntertialNavigation.m: 进行惯导的调用（所调用函数因工程需要涉暂保密）
- mainForPost.m: 获得相对姿态
